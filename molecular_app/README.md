# molecular_work_app

A starter Flutter app for organizing molecular biology workflows such as DNA cloning, transformation, transfection, plasmid prep, protein purification, and siRNA experiments.

## Getting started

This container does not have the Flutter SDK installed, so the project was scaffolded manually.

To run it locally:

```bash
flutter pub get
flutter run
```

## Structure

- `lib/main.dart`: app entry point and starter UI
- `pubspec.yaml`: package configuration
- `test/widget_test.dart`: basic widget test
